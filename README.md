# Privacy Policy for PouchCellInspector

**Last Updated:** March 10, 2026

## Introduction

The Pouch Cell Inspector mobile application is designed to assist users in inspecting and analyzing pouch cell batteries. This Privacy Policy explains how information is handled when you use our application.

We are committed to protecting user privacy and ensuring transparency regarding how information is processed when using the application.

## Information We Collect

PouchCellInspector is designed with privacy in mind. The application does not collect, store, or transmit personal data to external servers.

Any information generated while using the app remains on the user's device unless the user explicitly chooses to export or share that information.

### Data Generated Within the App

While using the application, users may generate certain types of information, such as:

- Inspection results
- Images captured for analysis
- Notes or observations entered by the user
- Analysis results produced by the on-device machine learning model

All of this information is stored locally on the user's device and is not transmitted to any external service by the application.

## On-Device Machine Learning

PouchCellInspector uses a machine learning model to assist with the inspection and analysis of pouch cell batteries.

Important privacy considerations:

- The machine learning model runs entirely on the user's device.
- Images or inspection data are processed locally on-device.
- No images or analysis results are sent to external servers.
- The machine learning model does not upload or transmit user data.

Because the model operates locally, all analysis occurs directly on the user's device, ensuring that inspection data remains private.

## Data Storage

All data generated within the application is stored locally on the device.

The application does not maintain external databases or remote servers for storing user data.

Users maintain full control over their data. Any stored information can be deleted by removing it within the application or uninstalling the app.

## Data Sharing

PouchCellInspector does not sell, rent, or share user data with third parties.

The application does not include:

- Third-party analytics services
- Advertising networks
- External tracking tools

Any data created in the application remains under the control of the user.

## Third-Party Services

The application does not rely on third-party services that collect user data.

All core functionality, including the machine learning analysis, runs locally on the device.

## Children's Privacy

PouchCellInspector is not directed toward children under the age of 13. The application does not knowingly collect personal information from children.

If you believe that personal information from a child has been provided through the app, please contact us so that appropriate action can be taken.

## Security

We take privacy and security seriously. Because the application does not transmit or store user data externally, the risk of unauthorized access through remote systems is minimized.

Users are responsible for maintaining the security of their own devices and controlling access to the information stored locally on those devices.

## Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. When updates are made, the "Last Updated" date at the top of this document will be revised.

Users are encouraged to review this Privacy Policy periodically.
